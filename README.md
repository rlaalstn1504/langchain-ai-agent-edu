# 2026년 전공분야 보수교육(융합) [IT직종]
## **LangChain을 활용한 AI 에이전트 개발 심화**

본 폴더는 **민간직업훈련기관 정보기술분야 교·강사 대상 보수교육(융합) 과정**
「LangChain을 활용한 AI 에이전트 개발 심화」의 교육자료 및 실습 예제 코드를 제공합니다.
최신 LangChain v1, LangGraph, Human-in-the-Loop, Agentic RAG, 테스트와 관측성까지 포함한 **실전형 AI 에이전트 개발 흐름**을 학습하는 것을 목표로 합니다.

## 실습 환경 준비

본 과정의 모든 실습은 **Python 3.10 기반 Conda 가상환경**에서 진행됩니다.
아래 절차를 완료하면 VS Code에서 노트북을 실행할 수 있습니다.

### 1. Anaconda 설치

운영체제에 맞는 Anaconda를 먼저 설치합니다.

Anaconda 공식 다운로드 페이지:  
https://www.anaconda.com/download/success

### 2. Conda 가상환경 생성 (Python 3.10)

```bash
conda create -n langchain_agent python=3.10
```

### 3. 가상환경 활성화

```bash
conda activate langchain_agent
```

### 4. 실습에 필요한 패키지 설치

본 저장소 루트 디렉터리에 포함된 `requirements.txt` 파일을 사용하여
실습에 필요한 라이브러리를 설치합니다.

```bash
pip install -r requirements.txt
```

### 5. VS Code 설치

본 과정의 실습은 VS Code에서 노트북을 열어 진행합니다.
VS Code가 설치되어 있지 않은 경우 아래 공식 페이지에서 설치합니다.

https://code.visualstudio.com/

### 6. VS Code에서 열기

```bash
code .
```

VS Code에서 노트북을 열고 커널을 `langchain_agent` 환경으로 선택합니다.

## 수료 평가 안내

수료 평가는 `최종과제.ipynb`의 **쇼핑몰 고객 문의 처리 에이전트** 실행 결과를 정리한 캡처 보고서로 진행합니다.

제출물은 **캡처 보고서 1개**입니다.

- 파일명: `2.수행평가 제출양식_본인이름.docx`
    - 본인 이름으로 변경해서 제출합니다.
    - 예시: `2.수행평가 제출양식_김민수.docx`
- 보고서 분량: 1~3쪽
- 실행한 노트북 파일은 선택 제출입니다.

보고서에는 아래 실행 결과를 포함합니다.

| 번호 | 필수 캡처 | 노트북 위치 |
|---|---|---|
| 1 | 주문 조회 Tool 실행 결과 | `3. 주문 조회 Tool 확인` |
| 2 | PII 처리 확인과 승인 대기 정보 | `4. PII 처리와 고객 문의 접수 요청` |
| 3 | 승인 후 재개 결과 | `5. 승인 후 재개` |
| 4 | 같은 `thread_id` 후속 질문 결과 | `6. 같은 thread_id로 후속 질문` |

총점은 100점이며, 수료 기준은 60점입니다.

| 항목 | 점수 | 채점 기준 |
|---|---:|---|
| 제출 형식 | 20점 | 지정된 파일명 형식의 DOCX 보고서 제출 |
| 주문 조회 Tool 확인 | 20점 | `lookup_order` 실행 결과 포함 |
| PII 처리 확인 | 20점 | 카드번호 원문이 입력 기록이나 Tool 인자에 그대로 남지 않음 |
| 승인 후 재개 확인 | 20점 | 승인 대기 정보와 승인 후 고객 문의 접수 결과 포함 |
| 후속 질문 및 설명 작성 | 20점 | 같은 `thread_id` 후속 질문 결과와 핵심 설명 포함 |
